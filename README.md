# Enjoeat_Server
Menu using tablet

## QR코드 스터디
- [QR_CODE](https://github.com/jungjai/QR_CODE)

## 배포

- AWS EC2 - 애플리케이션 서버
- AWS RDS - db 서버
- AWS S3 - 저장소 서버

- [vscode](https://code.visualstudio.com/) - 편집기
- [sublimetext](https://www.sublimetext.com/) - 편집기
- [Mysql](https://www.mysql.com/) - DataBase
